# Get Domain Backlinks


?> **POST** `{{{v1.server}}}/get_domain_backlinks`


Get the top backlinks for a single domain.

**Note:** Each backlink returned by this API endpoint consumes 1 credit.


### Authorization

You need an API key to access this API endpoint. Get your API key [here](https://keywordseverywhere.com/first-install-addon.html).


### Example API Calls

#### ** Bash **

- Urlencoded version

```shell script
params=()
params+=('domain=example.com')
params+=('num=5')
data=$(IFS='&' ; echo "${params[*]}")
curl -H 'Authorization: Bearer {{{v1.apiKey}}}' -H 'Accept: application/json' --data "$data" '{{{v1.server}}}/get_domain_backlinks'
```

- JSON version

```shell script
curl -H 'Authorization: Bearer {{{v1.apiKey}}}' -H 'Accept: application/json' -H 'Content-type: application/json' --data '{
    "domain": "example.com",
    "num": 5
}' '{{{v1.server}}}/get_domain_backlinks'
```


#### ** PHP **

```php
<?php

$parameters = [
  "domain" => "example.com",
  "num" => 5
];

}
```

#### ** Laravel **

```php
<?php

namespace App\Console\Commands;

use Illuminate\Support\Facades\Http;

 *
 * php artisan get-domain-backlinks

class GetDomainBacklinksCommand extends Command
{

    protected $signature = 'get-domain-backlinks';

    public function handle()
    {
        $response = Http::withToken('{{{v1.apiKey}}}')
            ->acceptJson()
            ->asForm()
            ->post('{{{v1.server}}}/get_domain_backlinks', [
                "domain" => "example.com",
                "num" => 5
            ]);

        try {
            echo $response->throw()->body();
        } catch (\Exception $e) {
            echo "An error occurred.\n\n" . $e->getMessage();
        }

        return 0;
    }
}
```

### Response HTTP Status Codes

| Status Code | Description                                                 |
|-------------|-------------------------------------------------------------|
| `200`       | A successful request                                        |
| `400`       | Submitted request data is invalid                           |
| `401`       | API key is missing or is invalid                            |
| `402`       | Account has an invalid subscription or insufficient credits |
